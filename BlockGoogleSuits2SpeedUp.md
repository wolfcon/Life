# 屏蔽 Google API / Recaptcha / Analytics 以加快网站访问速度

[TOC]

## Chrome

### 利用 AdblockPlus

添加以下屏蔽关键字

分别是

- Google 统计
- Google 人机认证
- Google API

```
||google-analytics.com/
||www.google.com/recaptcha/
||apis.google.com/
```

如果需要使用 `Google` 的 `API` 以及 `人机认证` 的话, 暂时禁用 `Ablock` 即可
