## Macros

```macro
#showtooltip
/dismount
/cancelaura slow fall
/cancelaura ice block
/cast frostbolt
```

```
/cancelaura mana shield
```

```
#showtooltip
/stopcasting
/cast counterspell
```

```
#showtooltip
/cast [button:2] amplify magic;dampen magic
```

```
#showtooltip
/cast [@mouseover] counterspell
```

```
#showtooltip
/cast [@mouseover] polymorph
```

```
#showtooltip
/cast [@mouseover] Fire Blast(Rank 1)
```

Dont toogle shoot

```
#showtooltip
/cast !shoot
```
